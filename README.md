# Student Performance Data Analysis

## Overview
This project performs a comprehensive analysis of student performance factors using Python. It explores how various demographics and preparation levels impact academic scores.

## Submission Details

- **Submission Date:** January 3, 2026
- **Project Type:** Individual Data Analysis Project

## Tools Used
- **Language:** Python
- **Libraries:** Pandas, Matplotlib, Seaborn, NumPy

## Analysis Phases
- **Data Acquisition:** Loading the dataset into a Pandas DataFrame.
- **Data Cleaning:** Handling missing values and formatting data types.
- **Analysis:** Calculating mean scores and identifying correlations.
- **Visualization:** Generating distribution plots and heatmaps to visualize performance trends.

## How to Run
1. Clone the repository.
2. Ensure you have the required libraries installed: `pip install pandas matplotlib seaborn`.
3. Run the analysis file (e.g., `python analysis.py` or open the `.ipynb` file).
